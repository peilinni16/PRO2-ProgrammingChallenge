Descripción detallada

Representa el conjunto de características y operaciones de los clústers.

Contiene operaciones consultoras que devuelven el identificador y la estructura del clúster; también una operación de escritura que imprime un clúster.

Dado que vamos a necesitar inicializar los clusters con elementos directos del Cjt_Especies, disponemos de una constructora inicial. Aparte, ofrece otra constructora que fusiona dos clusters, tomando por valor una nueva distancia a partir de la dada.
